## 🧠 Sobre o projeto

Este projeto além de ter sido meu primeiro contato com **React** no frontend e *FastAPI* no backend, foi criado para praticar conceitos importantes de desenvolvimento full stack, como:

- Consumo de APIs REST
- Integração entre frontend e backend
- Manipulação de dados em tempo real
- Criação de interfaces modernas

Os dados são obtidos através da API da CoinGecko.
